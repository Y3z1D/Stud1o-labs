# Stud1o-labs 🧪

**A multi-demo behavioral tracking and research laboratory.**

Built by [Y3z1D] under the TwinVault Technologies architecture, this repository holds a series of isolated, edge-deployed testing environments. These labs are designed to test custom behavioral tracking mechanics, telemetry data collection, and frontend cybersecurity configurations.

### 🌐 Live Environment
The complete multi-demo lab is live and deployed at the edge:
**[TwinVault Demos](https://demo.twinvault.ca)**

### 🏗️ Architecture & Tech Stack
* **Deployment Engine:** Cloudflare Workers (Edge Serverless)
* **Routing:** Custom asset routing directly from the `/Demo` directory
* **UI/UX Design:** High-contrast Dark UI featuring signature emerald green crystalline aesthetics
* **Version Control:** Standalone Git repository acting as a secure vault (disconnected from active CI/CD to prevent automated overwrites)

### 📂 Directory Structure
* `index.html` - The central dashboard and landing page routing engine.
* `Demo_1` through `Demo_7` - Completely isolated tracking environments for testing individual scripts and data collection methods.
